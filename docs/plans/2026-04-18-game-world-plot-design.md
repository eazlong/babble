# 游戏世界与情节设计文档

> **项目名称**：LinguaQuest RPG  
> **设计版本**：v1.0  
> **日期**：2026-04-18  
> **对应课标**：国内小学四年级英语教学大纲

---

## 1. 世界概述

### 主题：魔法学院冒险（Magic Academy Adventure）

玩家是一名新入学的魔法学院学生。在这个世界里，**英语就是魔法咒语** — 学会说英语就能施展魔法。学院有三处魔法区域，每处需要掌握特定的"英语咒语"才能解锁。

### 核心设定
- **语言 = 魔法**：英语表达是施展魔法的力量来源
- **精灵陪伴**：玩家有一个随身精灵 Spark（小灵）全程陪伴
- **徽章收集**：完成每个区域获得一枚魔法徽章
- **儿童友好**：无失败惩罚，只有鼓励和帮助

## 2. 角色设计

### 2.1 主角

| 项目 | 设计 |

### 2.2 NPC 阵容（共 5 个）

| # | NPC | 角色 | 场景 | 教授内容 | 动物形态 |
|---|-----|------|------|---------|---------|
| 1 | **Spark（小灵）** | 精灵教练 | 全流程 | 引导+教学+鼓励 | 发光小精灵 |
| 2 | **Oakley** | 森林守护者 | 精灵森林 | 问候语、颜色、数字 | 猫头鹰 |

- **Spark**：活泼热情，多用英语，辅以中文解释
- **Oakley**：智慧温和，语速较慢，像老爷爷
- **Bookmark**：博学稳重，喜欢用"慢慢来"的口头禅
- **Luna**：活泼可爱，同龄感，像好朋友
- **Petalia**：温柔耐心，充满母性关怀

## 3. 场景设计（3 场景）

### 3.1 精灵森林（Spirit Forest）

**主题**：初识魔法，学习基础

| 项目 | 内容 |
|------|------|
| 视觉风格 | 茂密森林、发光蘑菇、彩色魔法花、小溪流水 |
| 氛围 | 神秘但温馨，适合入门 |
| 时长 | 10-15 分钟 |

**学习内容**：
- 问候语：Hello! / Hi! / Good morning! / Good afternoon!
- 自我介绍：What's your name? I'm... / Nice to meet you!
- 数字 1-20：用于解开森林中的数字锁
- 颜色：Red, blue, yellow, green, orange, purple — 激活魔法花

**核心任务**：
1. 认识 Spark，说出自己的名字
2. 用颜色魔法激活三朵不同颜色的花
3. 用数字魔法打开 Oakley 的宝箱
4. 获得 **Forest Badge**

**任务流程**：
```
进入森林 → Spark 自我介绍 → 玩家语音输入名字 → Spark 教问候语
→ 遇到 Oakley → Oakley 发起森林试炼
→ 颜色任务：选择正确的颜色咒语激活花朵
→ 数字任务：数物品，说出正确数字解锁宝箱
→ 获得 Forest Badge → 前往图书馆
```

---

### 3.2 咒语图书馆（Spell Library）

**主题**：学习课堂用语和学习用品

| 项目 | 内容 |
|------|------|
| 视觉风格 | 古老图书馆、漂浮的书本、魔法书架、会动的羽毛笔 |
| 氛围 | 安静但有趣，充满知识的力量 |
| 时长 | 10-15 分钟 |

**学习内容**：
- 课堂指令：Stand up, sit down, open your book, close your book, listen, read, write
- 学习用品：pen, pencil, book, bag, ruler, eraser, desk, chair
- 动作动词：look, listen, read, write, draw, count, say

**核心任务**：
1. 帮 Bookmark 整理散落的魔法书（指令响应）
2. 与 Luna 进行对话练习
3. 合作完成"整理图书馆"任务（物品归类）
4. 获得 **Library Badge**

**任务流程**：
```
进入图书馆 → Bookmark 欢迎 → 教课堂指令
→ 指令游戏：Luna 说指令，玩家执行对应动作
→ 整理任务：把散落的学习用品放到正确书架
→ 与 Luna 自由对话练习
→ 获得 Library Badge → Spark 报告彩虹花园异常
```

---

### 3.3 彩虹花园（Rainbow Garden）

**主题**：探索自然，综合应用

| 项目 | 内容 |
|------|------|
| 视觉风格 | 五彩花园、浮动彩虹、蝴蝶飞舞、天气多变 |
| 氛围 | 活泼欢快，探索与冒险感 |
| 时长 | 15-20 分钟 |

**学习内容**：
- 天气：sunny, cloudy, rainy, windy, snowy, warm, cold, hot
- 动物：cat, dog, bird, fish, rabbit, butterfly, bee
- 植物：flower, tree, grass, leaf, fruit
- 方位：in, on, under, behind, next to
- 情感：happy, sad, tired, hungry, excited

**核心任务**：
1. 用天气咒语修复"天气水晶"
2. 找到迷路的小动物（方位指令）
3. 帮 Petalia 种植魔法花朵（颜色+植物词汇）
4. 获得 **Garden Badge**

**任务流程**：
```
进入花园 → Petalia 求助：天气魔法失控了
→ 天气任务：选择正确天气词修复水晶
→ 寻物任务：根据方位描述找到迷路动物
→ 种植任务：组合颜色+植物词汇种花
→ 获得 Garden Badge → 第一章完成
→ Spark 宣布：你已成为魔法学徒！
```

## 5. 游戏机制

### 5.1 经验值系统（LXP）

| 行为 | 奖励 |
|------|------|
| 正确回答 | 1-5 颗星星（根据发音准确度） |
| 首次完成任务 | 额外 10 星星奖励 |
| 连续正确 3 次 | 连击奖励 +5 星星 |
| 每个场景目标 | 积累 30 星星通关 |

### 5.2 徽章与称号

| 徽章 | 获得条件 | 解锁内容 |
|------|---------|---------|
| Forest Badge | 完成精灵森林 | 颜色魔法特效 |
| Library Badge | 完成咒语图书馆 | 书本跟随宠物 |
| Garden Badge | 完成彩虹花园 | 彩虹背景 |
| 全部集齐 | 3 枚徽章 | 魔法学徒称号 |

### 5.3 辅助与降级

| 情况 | 系统响应 |
|------|---------|
| 玩家长时间沉默（>10s） | Spark 主动提示，可用中文辅助 |
| 连续答错 3 次 | 自动降级：给出选项或更明显的提示 |
| 发音错误 | Spark 示范正确发音，鼓励重试 |
| 答对 | 即时正面反馈（动画+音效+星星） |
| 无负面反馈 | 永远不说"你错了"，只说"再试试" |

### 5.4 精灵教练 Spark 介入时机

| 触发条件 | 行为 |
|---------|------|
| 玩家沉默 >10 秒 | "Need help? Let me explain..." |
| 发音准确度 <40% | "Good try! Listen carefully: [正确发音]" |
| 任务完成 | "Great job! Excellent! You're amazing!" |
| 场景切换 | 介绍下一个场景的背景故事 |
| 玩家连续正确 | 逐步减少中文辅助，增加英语比例 |

## 6. 与四年级英语课标对应

| 课标要求 | 游戏中的体现 | 场景 |
|---------|------------|------|
| 能进行简单的问候与自我介绍 | 问候咒语 + 自我介绍咒 | 精灵森林 |
| 能识别和说出颜色 | 颜色魔法激活花朵 | 精灵森林 |
| 能认读数字 1-20 | 数字魔法开宝箱 | 精灵森林 |
| 能听懂课堂指令 | 课堂指令练习 | 咒语图书馆 |
| 能说出常见学习用品名称 | 学习用品词汇 | 咒语图书馆 |
| 能描述天气 | 天气水晶任务 | 彩虹花园 |
| 能说出常见动物名称 | 找迷路小动物 | 彩虹花园 |
| 能表达简单的方位 | 方位指令寻物 | 彩虹花园 |
| 能表达基本情感 | 与 NPC 对话互动 | 全流程 |
| 能进行简单的日常对话 | 与 Luna/Petalia 对话练习 | 图书馆/花园 |

### 语音交互四维度

| 维度 | 设计 |
|------|------|
| 听 | NPC 标准英语语音（TTS），语速较慢（~100 词/分钟），关键句重复播放 |
| 说 | 语音输入回应 NPC，Whisper ASR 判断发音准确度（1-5 星） |
| 读 | 游戏内文字中英双语，关键词汇带拼音式音节拆分 |
| 认 | 图像辅助理解，所有词汇配合图标/动画，不依赖翻译 |
